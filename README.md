# CLion UTF-8 C++ Exercises

This repository contains several standalone C/C++ console exercises.
Each `.cpp` file is built as its own executable target.

## Targets

- `record_app` from `1.cpp`
- `school_app` from `school.cpp`
- `test_app` from `test.cpp`
- `text_app` from `text.cpp`

## Encoding

- Source files are saved as UTF-8.
- Windows console input/output is switched to UTF-8 in code.
- CMake adds UTF-8 compiler flags for MinGW and MSVC-friendly UTF-8 settings.
